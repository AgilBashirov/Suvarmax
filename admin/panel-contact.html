Əlaqə və sosial şəbəkələr

Əlaqə (telefon, email, ünvan)

Sosial şəbəkələr (footer)

İkon haradan?

Yeni brend ikonları (Facebook, Instagram, YouTube və s.) üçün ən rahat mənbə Simple Icons vebsaytıdır — SVG yükləyib rəngi redaktə edə və ya birbaşa layihəyə əlavə edə bilərsiniz (MIT lisenziyası). Ümumi UI ikonları üçün Heroicons da istifadə oluna bilər.

Layihədə hazır nümunə fayllar assets/images/social/ qovluğundadır. İkon yolu sahəsinə yazın və ya «İkon yüklə» ilə həmin qovluğa yükləyin.

  • assets/images/social/facebook.svg
  • assets/images/social/instagram.svg
  • assets/images/social/linkedin.svg

SVG yükləyəndə server faylı avtomatik footer rənginə gətirir: #e5e7eb. PNG rəngi dəyişmir — ehtiyac varsa əvvəlcə SVG istifadə edin. Sosial sətirini silib «Əlaqəni saxla» dedikdə, həmin ikonun yerli faylı assets/images/social/ qovluğundan da silinir (standart facebook.svg, instagram.svg, linkedin.svg toxunulmur).
